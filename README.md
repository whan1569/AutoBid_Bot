# AutoBid Bot 프로젝트

이 프로젝트는 벡터DB 기반 아이템 군집화와 CTR/CVR 분석을 통한 광고비 최적화 시스템을 구현하는 연구 및 개발 프로젝트입니다.

## 프로젝트 개요

### 핵심 기능
1. **아이템 군집화**
   - 10개 기준에 따른 상품 분류 (카테고리, 용도/목적, 물리적 형태 등)
   - 벡터DB를 활용한 유사 상품 그룹화
   - 군집별 특성 분석

2. **CTR/CVR 분석**
   - 군집별 클릭률(CTR)과 전환율(CVR) 분석
   - 트렌드 기반 광고비 최적화 전략 수립
   - 실시간 성과 모니터링

3. **동적 입찰 시스템**
   - 실시간 입찰가 조정 알고리즘
   - CTR/CVR 추세 기반 광고비 최적화
   - 백테스트를 통한 전략 검증
